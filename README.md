# OnlineSinav

<p>Kullanıcıların sınav soruları hazırlayabilme ve hazırlanan sorular ile sınav olabilme özelliklerine sahiptir.</p>
<p>Spring MVC ve MySQL teknolojileri kullanılarak oluşturulmuştur.</p>
